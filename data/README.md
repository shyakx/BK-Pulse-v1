# Data Directory

This directory contains datasets used for model training and analysis.

## Structure

- `raw/` - Original, unprocessed datasets
- `processed/` - Cleaned and preprocessed datasets ready for model training
- `models/` - Trained model files (if saved locally)

## Usage

Place your training datasets in the appropriate subdirectory based on their processing stage.

## Notes

- Large files should be added to `.gitignore` if using version control
- Document data sources and preprocessing steps in the relevant subdirectory
